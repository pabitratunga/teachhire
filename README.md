# TeachHire

India's premier platform for academic job postings.

## Tech Stack
- React 18 + Vite
- Tailwind CSS
- Lucide Icons

## Local Development
```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Push this repo to GitHub
2. Go to vercel.com → New Project → Import from GitHub
3. Select this repo — Vercel auto-detects Vite, no config needed
4. Click Deploy
